# E-Commerce Customer Behavior Analysis (Power BI)

## Project Overview
This project analyzes customer purchasing behavior for an e-commerce business using **SQL Server** as the data source and **Power BI** for visualization.  
The goal is to uncover actionable insights related to **customer value, retention, product performance, revenue distribution, and payment behavior**.

---

##  Data Source
- **Database:** SQL Server
- **Tables Used:** customers, orders, order_items, products, payments
- **Order Filter:** Completed orders only

---

## Key Business Questions Answered
- Who are the highest-revenue and most valuable customers?
- How many customers are one-time vs repeat buyers?
- Which products generate the most revenue and volume?
- Which customers are most valuable based on RFM analysis?
- What is each customerâ€™s lifetime value (CLV)?
- Which payment methods are most popular and profitable?

---

## Core Metrics (DAX Measures)
- Total Revenue  
- Total Orders  
- Average Order Value (AOV)  
- Revenue per Customer  
- One-Time vs Repeat Buyers  
- RFM (Recency, Frequency, Monetary)  
- Customer Lifetime Value (CLV)  
- Product Revenue & Units Sold  

(All metrics are calculated dynamically using Power BI DAX.)

## Tools & Technologies

## Key Insights
- Identified high-value (VIP) customers for targeted retention.
- Highlighted best-selling and high-revenue products.
- Revealed strong repeat-purchase behavior among top customers.
