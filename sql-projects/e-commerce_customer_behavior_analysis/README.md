E-Commerce Order & Customer Behavior Analysis (SQL)

Project Overview
This project analyzes customer purchasing behavior and revenue patterns for an e-commerce business using SQL Server.
The goal is to understand customer value, repeat purchasing behavior, and top-performing products to support data-driven business decisions.

Dataset
The analysis uses five relational tables:


customers


orders


order_items


products


payments


Data was imported from CSV files into SQL Server Management Studio (SSMS) and relationships were created using primary and foreign keys.


Key Business Questions Answered


Who are the highest-value customers?


How many customers are repeat vs one-time buyers?


What is the Customer Lifetime Value (CLV)?


Which products and categories generate the most revenue?


What payment methods are most commonly used?


SQL Skills Demonstrated


Multi-table JOINs


Aggregations (SUM, COUNT, MAX)


Customer segmentation (RFM analysis)


Business-driven analytical queries


Clean, well-commented SQL scripts



Dashboard
Insights from the SQL queries were visualized in Power BI, including:


Revenue per customer


Repeat vs one-time buyers


Top products by revenue


Customer value distribution
